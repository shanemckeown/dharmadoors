## Vision

### 10x Check
The 10x version of SanghaMap is the authoritative, community-verified source of truth for Buddhist community data globally. Think Wikipedia for Buddhist centers: living data, cross-tradition, ethically transparent, with every center verified by practitioners who've actually visited.

### Strategic Pivot: OSM as Primary Data Source
During CEO review, the outside voice challenge revealed that OpenStreetMap contains 108,000+ Buddhist places of worship (vs the scraper's theoretical 5,000-40,000). 24,000 are already classified by tradition/denomination. This data is free, legal, and aligned with the dana model. The data strategy flipped:
- **OSM as primary** — 108K+ centers, free, legal, pre-classified
- **Google scraper as enrichment** — adds reviews, phone, website, opening hours to OSM records
- **Community submissions** — fills gaps neither source has

### Delight Opportunities Surfaced
1. "First Visit" tradition guides — remove the anxiety of walking into an unfamiliar space
2. Tradition explainer tooltips — make cross-tradition discovery useful for newcomers
3. Shareable center cards (OG images) — viral infrastructure
4. Statistics page — credibility, SEO, coverage dashboard
5. "Visited" tracking — personal dimension, "my Buddhist journey"

### Eng Review Decisions
- **Per-country Overpass queries** — query via `area["ISO3166-1"="XX"]` to avoid global timeout
- **No country-based tradition inference** — untagged centers stay "Other" to preserve cross-tradition discovery (a Thai practitioner searching for Zen should find the Zen centers, not have them hidden under "Theravada")
- **500 marker cap removed** — rely on react-leaflet-cluster, add cap only if mobile testing shows issues
- **Vitest added** — unit tests for pipeline, adapter, traditionsData
- **Critical gaps to handle in implementation:** GeoJSON fetch errors need try/catch with user-visible fallback; localStorage access needs try/catch for private browsing

## Design Specifications (added by /plan-design-review 2026-03-26)

### Layout — Desktop (1024px+)

First-time visit: centered search overlay on map ("Find sangha near you") with Near Me button below. After search, sidebar (240px) slides in on left with filters; map takes remaining width.

Sidebar contains: search input (moved from overlay), tradition chips (multi-select color-coded pills), Near Me button, coverage stats, visited toggle. Map is the primary element (~75% of viewport).

### Layout — Mobile (< 768px)

Full-screen map. Search bar floating at top (fixed). Filter icon (bottom-right FAB) opens bottom sheet with tradition chips (horizontal scroll) + Near Me. Detail popup → full-width bottom sheet (60% height, draggable).

### Layout — Tablet (768-1023px)

Sidebar collapses to bottom sheet (40% height, draggable). Map full width. Tradition chips scroll horizontally.

### Map Markers

Custom SVG teardrop pins colored by tradition (using BRAND_GUIDELINES tradition colors). "Other/Unknown" = gray (#6B7280). Clusters show count, colored by dominant tradition (mixed = Dharma Tan #A09078). Cluster sizes: small (2-10), medium (11-50), large (50+).

Visited indicator: small saffron checkmark dot in bottom-right corner of pin.

### Tradition Filter Chips

Multi-select toggle pills. Each chip shows tradition color background + white text. 8px vertical padding, 16px horizontal, 999px border-radius. Active chips are filled; inactive are outlined. Tap to toggle on/off.

### Detail Popup (desktop = Leaflet popup, mobile = bottom sheet)

Content hierarchy:
1. Center name (H3, Source Sans 3, bold)
2. Tradition badge (colored chip)
3. Address (full, multi-line)
4. Contact: phone, website (if available)
5. Rating (if from enrichment data, otherwise omit)
6. [Mark as Visited] toggle
7. Expandable: First Visit Guide (chevron toggle, 3-4 sentences, friendly tone)
8. Expandable: About [Tradition] (one-paragraph tooltip)
9. Expandable: Ethical Concerns (amber warning triangle, only if flagged — amber left-border accent, NOT red)
10. Action buttons: [Share] [Directions]

Missing fields show "—" not blank space.

### Statistics Page (/sanghamap/stats)

Single hero number layout (NOT 3-column grid):
- "108,232" in Cormorant Garamond 64px, #3D3629
- "Buddhist centers worldwide" supporting text
- "across 42 countries • 9 traditions" secondary

Horizontal bar chart: tradition breakdown with tradition-colored bars. Unclassified shown in gray. Top countries in asymmetric grid (not equal columns).

Source attribution: "Data from OpenStreetMap • Last updated: [date]" with link to contribute.

### OG Card (1200x630px)

Cream background. Center name in Cormorant Garamond (large). Tradition chip with color. Mini map snippet showing the pin location. DharmaDoors logo small in bottom-right corner.

### Interaction States

| Feature | Loading | Empty | Error | Success |
|---|---|---|---|---|
| Map initial | Skeleton with pulsing cream bg | N/A | "Couldn't load map tiles" + retry | Map + tiles visible |
| GeoJSON fetch | Subtle spinner in map corner | "No centers in this region — pan to explore" | "Couldn't load centers — try panning again" | Markers appear with cluster animation |
| Tradition filter | Instant (client-side) | "No [tradition] centers here. Try zooming out or removing filters." | N/A | Markers filter, count updates |
| Search | "Searching..." in input | "No results for '[query]'. Try a city name." | "Search unavailable — try again" | Map pans to location |
| Near Me | "Finding your location..." | N/A | "Location denied. Enter your city instead." | Map pans + zooms |
| Visited toggle | Instant | N/A | Silent graceful fail (private browsing) | Checkmark animation |

### Accessibility

- Touch targets: 44px minimum (chips, buttons, Near Me)
- Keyboard nav: Tab through search → Near Me → chips → map markers
- Screen readers: map `role="application"` + `aria-label="Buddhist center map"`, markers get `aria-label="{name}, {tradition}, {city}"`, chips get `role="checkbox"` + `aria-checked`
- Color contrast: WCAG AA (4.5:1) for all text
- Reduced motion: `prefers-reduced-motion` disables map animations and cluster transitions
- Focus indicators: 2px saffron outline
